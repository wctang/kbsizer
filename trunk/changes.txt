    - Win + NumPad: Move window with Numpad.
      Win + NumPad 0 :  Maximize / Restore
      Win + NumPad .  :  Minimize 
    - Win + Ctrl + (2-keypad number sequence) : Resize window to fit specified area. 
      First entered number sets the start point and second number sets the end point.
